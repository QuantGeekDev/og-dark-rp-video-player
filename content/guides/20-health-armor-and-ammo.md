# Health, Armor, and Ammo

## Quick Start

- Medic roles can heal with medical tools.
- Health kits and armor vests can be bought or produced through support systems.
- Health and armor labs create pickups over time.
- Nanite ammo systems supply special ammo boxes, shipments, or dispensers.
- Use pocket storage for supported pickups and boxes.

## Details

- Health and armor values are clamped by server rules.
- Labs and dispensers use intervals, storage, ownership, and floating status.
- Some support items are sold in shop categories and taxed normally.
- Skills can improve support-lab or pickup value when those hooks are active.
- Police and combat roles may start with special support gear.

## Gotchas

- Pickups are not infinite.
- Labs need safe placement like other machines.
- Ammo type matters. A gun may not use the box you found.
- Support gear does not override job authority or combat rules.
