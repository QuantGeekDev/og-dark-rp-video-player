# Hitman Contracts

## Quick Start

- Switch to Hitman.
- Open the hitman contract UI when available.
- Accept or create valid contracts through the server flow.
- Complete the target under the contract rules.
- Expect police and players to react to public violence.

## Details

- Contract state is tracked by the host.
- Pricing, escrow, settlement, and target validity are handled by hitman services.
- The Hitman job starts with scoped combat gear.
- Contract work should be focused on valid targets, not random deathmatch.

## Gotchas

- A weapon does not make every kill legal.
- Targets can become invalid if state changes.
- Staff rules still apply to contract roleplay.
- Do not rely on verbal side deals when the contract system has a formal flow.
