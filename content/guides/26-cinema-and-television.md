# Cinema and Television

## Quick Start

- Cinema Owner can run public screenings.
- TVs can be bought from supported shop paths.
- Press E on your TV to open the remote popup.
- Paste a supported YouTube link or direct media if the backend allows it.
- Stop or retune media when the scene changes.

## Details

- YouTube playback uses a hosted kiosk page rendered inside S&box.
- The game stores only normalized playback state, ownership, range, and permissions.
- The kiosk handles the actual YouTube iframe.
- Owners and admins control tuning.
- Cinema screens can wrap TV playback for larger venues.

## Gotchas

- Not every video URL is supported.
- Playlists, recommendations, and drift are intentionally restricted.
- Do not play content that violates server rules.
- Media can fail if the external service or kiosk is unavailable.
