# Casino and Coinflip

## Quick Start

- Casino Owner can place casino machines when allowed.
- Slot machines and poker tables use their own interaction panels.
- Coinflip is available from the Dashboard shortcut when synced.
- Set stakes you can afford to lose.
- Fair-play systems do not get better odds from skills.

## Details

- Casino machines are host-authoritative and use policy/math helpers.
- Coinflip offers can be public or targeted depending on the active UI.
- Poker uses card, deck, hand, rake, and table rules.
- Slots use defined machine policy and payout math.
- Skills can award capped participation XP, but not odds boosts.

## Gotchas

- Gambling is risk, not guaranteed income.
- Do not advertise false odds to other players.
- Casino objects can still be physical world entities that need safe placement.
- A lucky streak is not a balance promise.
