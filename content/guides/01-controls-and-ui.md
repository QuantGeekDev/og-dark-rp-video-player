# Controls and UI

## Quick Start

- Press Tab to open the main Dark RP menu.
- Press E on doors, NPCs, machines, vehicles, and other interactables.
- Use the weapon selector to swap tools and weapons.
- Press F1 for yes and F2 for no when a vote appears.
- Press Escape for the pause menu.

## Details

- The Dashboard shows your name, job, wallet, salary, tax, treasury, and legal status.
- Jobs changes, shops, skills, and waypoints all live in the Tab menu.
- World prompts usually mean the object can be used with E.
- Toasts and HUD banners tell you when money, law, votes, or machine events happen.
- Some tools have primary, secondary, and reload actions. Read the tool info panel when available.

## Gotchas

- If typing in a text box, Tab/Escape behavior may be captured by the input.
- Some panels only show for the right job, like Police or Mayor.
- Use waypoints when you cannot find an NPC or city service.
