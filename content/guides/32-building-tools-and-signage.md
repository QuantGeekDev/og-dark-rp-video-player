# Building Tools and Signage

## Quick Start

- Use the Physgun to move and freeze props you can control.
- Use the Toolgun for constraints, material, fading doors, buttons, keypads, signage, and other build tools.
- Build inside valid spaces and keep paths usable.
- Use signage tools for readable roleplay labels.
- Clean up abandoned or broken builds.

## Details

- Ownership checks decide who can modify a prop.
- Toolgun modes have local settings and server validation.
- Fading doors and keypads are base tools, not decoration only.
- Save and dupe systems preserve sandbox builds when allowed.
- Admin cleanup tools can remove harmful or abandoned objects.

## Gotchas

- Do not block spawn, streets, NPCs, or required doors.
- Tool settings can persist between uses.
- Some props are not valid tool targets.
- Staff can remove builds that harm roleplay or performance.
