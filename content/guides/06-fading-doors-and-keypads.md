# Fading Doors and Keypads

## Quick Start

- Use the Toolgun Fading Door tool to convert a prop.
- Set a bound key for the door.
- Place a button or keypad with the same bound key.
- Use the keypad by looking at it, typing the code, and pressing E or Enter.
- Owners can trigger their own bound keys from the keyboard.

## Details

- Signals are scoped to the owner, so two players can use the same key safely.
- Doors can toggle or pulse open depending on their fade duration.
- Keypads accept digit codes and can hide entry with secure display.
- The faded state is ghostly instead of fully invisible.
- Lockpicks can breach fading doors when the raider has the right job/tool.

## Gotchas

- Button behavior comes from the door settings, not a separate button toggle.
- Looking away releases keypad focus.
- Rebinding keys will not fire the door while the bind control is capturing input.
- Empty or invalid keys disable signal activation.
